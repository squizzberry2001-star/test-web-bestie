Store Master

Dropdown store akan tampil sebagai kode toko + nama store.

0 store

Auditor Master

Nama auditor tampil di dropdown interaktif. NIK disimpan untuk hasil sheet report.

0 auditor

Format auditor yang direkomendasikan: kolom NIK dan Nama Auditor. Saat import, master auditor akan diganti mengikuti file Excel.

Level Master

0 level
