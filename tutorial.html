Panduan Setup & Deploy

Halaman ini sekarang dipakai sebagai panduan teknis setelah upgrade. Fitur export PPT sudah dibuat langsung dari halaman report, jadi kamu tidak perlu lagi mengunggah PDF ke layanan pihak ketiga untuk membuat slide presentasi.

Yang berubah

  • Export PPT sekarang langsung dibuat dari browser lewat tombol Download PPT.
  • Config frontend dipusatkan di file app-config.js.
  • Autosave dipisah antara data form dan data media agar input terasa lebih ringan.
  • Apps Script backend sudah mendukung upload report, analytics, tracking usage, dan dashboard usage.

Script Properties Apps Script

  1. Buka project Apps Script.
  2. Masuk ke Project Settings → Script Properties.
  3. Tambahkan properti berikut:
    • ROOT_FOLDER_ID
    • SPREADSHEET_ID
    • SHEET_NAME (opsional, default Sheet1)
    • TIMEZONE (opsional, default GMT+7)
    • USAGE_DASHBOARD_CODE untuk membuka panel activity

File yang perlu kamu update

  1. Deploy file apps_script_check.js ke Apps Script.
  2. Update URL Web App di app-config.js bila endpoint berubah.
  3. Upload folder web statis ini ke hosting kamu seperti Vercel / Netlify.

Contoh properti

ROOT_FOLDER_ID=folder_google_drive_kamu
SPREADSHEET_ID=spreadsheet_google_sheet_kamu
SHEET_NAME=Sheet1
TIMEZONE=GMT+7
USAGE_DASHBOARD_CODE=kode_dashboard_rahasia
